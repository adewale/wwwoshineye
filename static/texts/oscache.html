The Abode::Using OS Cache

Decide on caching approach
Choose keys
Choose when and where entries will be flushed
Register filters in web.xml
Edit oscache.properties accordingly.
Deploy.

If you aren't using a pure Model 2 architecture (because your JSPs contain code that interacts with the model) then you should read this tutorial on using the "<cache>" tag to perform caching at the JSP level.
And of course there's always the possibility of using OS Cache as a generic caching library. There's nothing to tie it to JSPs or Servlet Filters.
