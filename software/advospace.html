The Abode::AdvoSpace

A simple proof of concept showing that Advogato's diary ratings in combination with the certification system can be mined for interesting correlations between people. It's written in Python and can be downloaded here.

Instructions on how to use it are at the top of the file. An example of the kind of output it produces is here.

This exists partly as an effort to make concrete my own random musings about trust metrics and partly to answer the question "what are trust metrics useful for?"
